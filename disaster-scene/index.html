DISASTER
SCENE

大场面不是成本堆砌。每个镜头都必须服务核心冲突、行动压力和情绪递进。

CORE IDEA 灾难级场景的根因不是画面差，而是目标、节奏、博弈和情绪互相脱节。 KEY QUESTION 观众此刻到底在担心什么？角色此刻到底必须解决什么？
诊
场景医生方法卡
DIAGNOSIS
先问这场戏是不是有效冲突

无效冲突的症状是：设定很危险，画面不危险；台词很着急，行动不着急；目标说得很大，场面只在耍帅。

1
Goal

一句话写清楚当下要解决的问题：救人、推翻、撤离、拖延还是反杀。

2
Stakes

不要只靠旁白喊危急，要拍出失败后马上发生的具体代价。

3
Pressure

行动必须被时间、距离、敌人、空间或人群失控持续挤压。

4
Payoff

高潮要兑现“拼尽全力才赢”，不能靠主角光环或敌人放水收场。

FOUR FAILURES
灾难级场景的四个断裂
A 冲突和目标脱节

救父戏变成展示能打，单挑戏变成展示光环，逃难戏变成展示团结。观众不知道该紧张哪件事。

场景目标 ≠ 展示主题
B 节奏和情绪脱节

该快时喊口号，该打时插回忆，该逃时慢慢行礼。情绪刚起来，就被错误节拍截断。

C 冲突没有博弈

敌军像纸糊，反派像放水，追兵永远追不上。没有“差一点失败”，就没有情绪波动。

REPAIR KIT
把大场面修成可感知压力

修改不是加更多特效，而是让观众看见危险、看见阻碍、看见角色用行动付出代价。

RESCUE 救父戏改成闯三道防线加限时处决，每一道都必须差一点失败。
DUEL 单挑戏删无关回忆，做成商王占上风、姬发濒死、借力反杀的节拍。
EXODUS 逃难戏用追兵逼近、人群摔倒、亲人失散和路线堵塞制造恐慌感。
SCENE BRIEF

按“场景目标 → 冲突节拍 → 镜头证据”写。每个镜头都回答：它让目标更难了吗？让情绪更强了吗？

大场景的情绪不是喊出来的，是被细节一点点逼出来的。

CONTROL PANEL
张弛有度，不是张弛无度
该快

救人、逃难、敌人逼近时，少解释，靠动作推进。

该慢

关键选择、代价落下、关系裂变时，让观众看清。

坏慢

生死时速里停下来喊话、寒暄、摆姿势。

坏快

高潮对决里乱切回忆，打断刚建立的压力。

DOCTOR'S RULE

先铺垫伤害，再发动行动；先建立优势方，再制造反制；先让角色濒临失败，再允许胜利成立。

NO LOGIC, NO EMOTION
FINAL CHECK

删掉不服务目标的耍帅、口号和空特写。留下会改变局势、增加压力、推动情绪的镜头。
