PURSUIT
SCENE

追杀戏不是一直跑，而是持续制造、维持、升级并在高潮处释放紧迫感。

CORE IDEA 每一拍都要让观众感到距离更近、空间更小、时间更少、选择更坏。 KEY QUESTION 危机有没有在观众疲劳之前升级？释放有没有发生在高潮点？

追杀戏方法卡
THE ENGINE
紧迫感的四段循环

一场好追杀要先让观众理解危险等级，再用节拍把冲突推进成一轮轮更坏的局面。

1
Reset

上一段危机解除，给一口气；追杀者重新出现，危机重启。

2
Close

压缩双方相对距离或速度差：从走到跑，从隔门到同室。

3
Upgrade

在疲劳前升级危机：更小空间、更快敌人、更失控载具。

4
Release

高潮处迅速释放，让观众意犹未尽，而不是把刺激拖到发麻。

PRESSURE DESIGN
怎么维持并升级危机
A 先铺垫追杀者

追杀者越早被证明危险，登场时越省解释。杀伤力、恢复力、速度、形态变化都能变成紧迫感本金。

危险铺垫 → 登场即压迫
B 压缩距离与空间

从长廊到电梯，从停车场到车身，把冲突推到更小、更难逃、更不可控的空间。

远距离追 → 近身攻防 → 狭小空间
C 快速换地点

短时间内切换长廊、电梯、停车场、车上、公路，既换视觉信息，也让观众注意力不断刷新。

BEAT GAME
把奔跑写成你来我往

追杀戏不能只有逃。每个节拍都要有“我方动作、敌方反制、局势变化”，让场景形成博弈感。

BEAT 我方开枪逃跑，敌方加速追击并还击。
BEAT 我方进电梯防守，敌方破门或从顶部进攻。
BEAT 我方抢车撤离，敌方抓车尾，把危机移到车身。
PARALLEL CLOCK

最有效的结构是并行倒计时：一边开锁、做炸弹、搬障碍；一边敌人破门、靠近、压迫。

紧迫感来自“事情还没做完，但敌人已经到了”。

RED HOOK
红钩：让角色期望落空
弱设计

发现锁打不开，下一秒就打开；敌人破门也没造成新压力。

强设计

开锁需要时间，敌人正在破门，炸弹还可能失灵。

重复障碍

连续三道打不开的门，会暴露想象力不足。

新障碍

每次升级都换压力来源：速度、空间、奇观、倒计时。

RED HOOK FORMULA

角色以为炸弹会准时爆炸，就让插头被踩掉；角色以为门能及时打开，就让敌人先闯进同一空间。

EXPECTATION FAILS, PRESSURE SPIKES
CHECKLIST

删掉追杀中的闲聊；不要重复同一种障碍；让配角各司其职；高潮释放后留下下一场危机的余味。
